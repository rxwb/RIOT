CoAP Networking

CoAP (Constrained Application Protocol) is a protocol to implement REST APIs for constrained devices and networks. It is designed for IoT applications and is similar to HTTP but much lighter.

This guide walks you through creating a simple CoAP application using RIOTâ€™s gcoap module. You will build both a server that responds to requests and a client that sends requests, demonstrating basic CoAP communication.
